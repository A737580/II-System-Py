Дебаг
    Если хочешь ещё больше контроля, можно окружить код в try/except и показывать ui.notify() с ошибкой.
Позиционированые объектов
    Row / Column (flex-контейнеры) → удобны для адаптивности.
    "
    with ui.row().classes("gap-4"):
    ui.button("Кнопка 1")
    ui.button("Кнопка 2")
    "
CSS классы (через .classes("...")) → можно управлять почти как в CSS.
    "
    ui.label("Привет").classes("absolute top-[100px] left-[200px]")
Свой CSS
    "
    with ui.element('div').classes('relative w-full h-screen'):
    ui.label("A").classes('absolute top-[50px] left-[100px]')
    ui.label("B").classes('absolute top-[200px] left-[300px]')
    "

Совет: для быстрого прототипа можно ставить absolute top-[Xpx] left-[Ypx].
Для боевого интерфейса лучше комбинировать с row/column, чтобы интерфейс был адаптивный.
